## 🔐 관리자 로그인

### 접속 방법

**방법 1: 랜딩 페이지에서**
```
1. 랜딩 페이지(landing.html) 접속
2. 하단 "👑 관리자 로그인" 링크 클릭
3. auth.html로 자동 이동
4. "관리자" 탭 클릭
```

**방법 2: 직접 접속**
```
URL: https://your-domain.com/auth.html
→ "관리자" 탭 클릭
```

### 로그인 정보
```
아이디: [보안상 비공개]
비밀번호: [보안상 비공개]
```

### 로그인 후
- 자동으로 `index.html`로 이동
- 상단 메뉴에 **"관리자"** 탭 표시
- 일반 회원은 볼 수 없는 기능

## 👥 회원 승인 시스템

### 1. 승인 대기 회원 확인

**위치**: 관리자 페이지 → "승인 대기 중인 회원" 섹션

**표시 정보**:
- 프로필 사진 (Google 로그인 시)
- 이름
- 이메일
- 가입일

### 2. 회원 승인

```
절차:
1. 승인 대기 목록에서 회원 확인
2. "✅ 승인" 버튼 클릭
3. 확인 창 → "예" 클릭
4. 완료! 회원이 로그인 가능하게 됨
```

**효과**:
- `is_approved: true` 설정
- 회원이 로그인 가능
- 모든 기능 사용 가능

### 3. 회원 거부

```
절차:
1. 승인 대기 목록에서 회원 확인
2. "❌ 거부" 버튼 클릭
3. 확인 창 → "예" 클릭
4. 완료! 회원 계정 삭제됨
```

**효과**:
- 계정 완전 삭제
- 로그인 불가
- 재가입 필요

## 👑 관리자 권한 관리

### 1. 승인된 회원 목록

**위치**: 관리자 페이지 → "승인된 회원 관리" 섹션

**표시 정보**:
- 프로필 사진
- 이름 + 관리자 배지(👑)
- 이메일
- 가입일
- 마지막 로그인

### 2. 일반 회원 → 관리자 승격

```
절차:
1. "승인된 회원 관리" 섹션 확인
2. 원하는 회원 찾기
3. "👑 관리자로" 버튼 클릭
4. 확인 창:
   "XXX 님을 관리자로 승격하시겠습니까?
    ⚠️ 관리자는 모든 회원을 관리하고 시스템을 제어할 수 있습니다."
5. "예" 클릭
6. 완료! 👑 배지 표시됨
```

**관리자 권한**:
- ✅ 회원 승인/거부
- ✅ 다른 회원을 관리자로 승격
- ✅ 관리자 권한 제거
- ✅ 회원 삭제
- ✅ 운영자 추천 설정
- ✅ API 키 관리
- ✅ 제외 채널 관리

### 3. 관리자 → 일반 회원 강등

```
절차:
1. "승인된 회원 관리" 섹션 확인
2. 관리자 권한이 있는 회원 찾기 (👑 배지 표시)
3. "↓ 일반 회원으로" 버튼 클릭
4. 확인 창:
   "XXX 님의 관리자 권한을 제거하시겠습니까?"
5. "예" 클릭
6. 완료! 일반 회원으로 변경
```

**효과**:
- 관리자 탭 사라짐
- 관리자 기능 사용 불가
- 일반 회원 권한만 유지

### 4. 회원 삭제

```
절차:
1. "승인된 회원 관리" 섹션
2. 삭제할 회원 찾기
3. "🗑️ 삭제" 버튼 클릭
4. 확인 창:
   "XXX 님을 삭제하시겠습니까?
    ⚠️ 이 작업은 되돌릴 수 없습니다."
5. "예" 클릭
6. 완료! 계정 완전 삭제
```

**⚠️ 경고**:
- 되돌릴 수 없음
- 로그인 불가
- 모든 데이터 삭제
- 재가입 필요

## 🌟 운영자 추천 관리

### 1. 작품 검색

**위치**: 관리자 페이지 → "운영자 추천 & 안전도 평가" 섹션

```
절차:
1. 검색창에 작품 제목 입력
2. Enter 또는 "🔍 검색" 버튼 클릭
3. 검색 결과 표시
```

### 2. 운영자 추천 설정

```
절차:
1. 검색 결과에서 작품 찾기
2. "👑 운영자 추천" 버튼 클릭
3. 완료! 메인 페이지에 노출
```

**효과**:
- 홈 탭 최상단 배치
- 금색 테두리 + 보라색 펄스 배지
- "명작 컬렉션"에 우선 노출
- S등급 자동 부여

### 3. 운영자 추천 해제

```
절차:
1. 검색 결과에서 추천 중인 작품 찾기
2. "❌ 추천 해제" 버튼 클릭
3. 완료! 메인 페이지에서 제거
```

**추천 기준**:
- ✅ 저작권 안전성 100% 확인된 작품
- ✅ 쇼츠 제작에 적합한 작품
- ✅ 실제 제작 경험으로 검증된 작품
- ✅ 대중적으로 인기 있는 작품

## 🔑 API 키 관리

### 1. TMDB API 키

**용도**: 영화/드라마 정보 자동 수집

```
절차:
1. https://www.themoviedb.org/settings/api 접속
2. API 키 발급
3. 관리자 페이지 → "API 키 관리" 섹션
4. "TMDB API 키" 입력
5. "저장" 버튼 클릭
```

### 2. YouTube API 키

**용도**: 쇼츠 개수 자동 수집, 안전도 분석

```
절차:
1. https://console.cloud.google.com/apis/credentials 접속
2. YouTube Data API v3 활성화
3. API 키 발급
4. 관리자 페이지 → "YouTube API 키" 입력
5. "저장" 버튼 클릭
```

### 3. KMDB API 키

**용도**: 한국 고전영화 (2000년 이전) 수집

```
절차:
1. KMDB API 키 발급 (선택)
2. 관리자 페이지 → "KMDB API 키" 입력
3. "저장" 버튼 클릭
```

## 🚫 제외 채널 관리

### 1. 위험 채널 추가

**용도**: 저작권 위험이 있는 YouTube 채널 차단

```
절차:
1. "제외 채널 관리" 섹션
2. "채널명 또는 ID" 입력
3. "추가" 버튼 클릭
4. 완료! 해당 채널의 쇼츠는 AI 분석에서 제외
```

**자동 차단되는 채널**:
- 🔴 PLAYY MOVIE (공식 채널)
- 🟠 웃웃티비
- 🟠 1분 드라마
- 🟠 tvN D ENT
- 🟠 DRAMA Voyage
- 🟠 SBS Catch

### 2. 채널 삭제

```
절차:
1. 제외 채널 목록 확인
2. "🗑️ 삭제" 버튼 클릭
3. 완료! 해당 채널 다시 분석 가능
```

## ⚠️ 주의사항

### 관리자 권한 관리
- ⚠️ 자신의 관리자 권한을 스스로 제거할 수 있으므로 주의
- ⚠️ 다른 관리자가 나의 권한을 제거할 수 있음
- ⚠️ 최소 1명의 관리자는 항상 유지해야 함

### 회원 관리
- ⚠️ 회원 삭제는 되돌릴 수 없음
- ⚠️ 회원 거부 = 계정 삭제
- ⚠️ 승인 취소는 불가능 (삭제만 가능)

### 운영자 추천
- ⚠️ 저작권 안전성을 반드시 확인
- ⚠️ 실제 제작 경험이 있는 작품만 추천
- ⚠️ 너무 많은 작품을 추천하면 효과 감소

### API 키
- ⚠️ API 키는 안전하게 보관
- ⚠️ YouTube API는 하루 10,000 유닛 제한
- ⚠️ TMDB API는 무료 (하루 40,000 요청)

## 📞 문제 해결

### 로그인 안 될 때
```
1. 아이디: [보안상 비공개]
2. 비밀번호: [보안상 비공개]
3. 브라우저 캐시 삭제
4. 시크릿 모드로 접속
```

### 회원 승인 안 될 때
```
1. F12 콘솔 확인
2. 네트워크 오류 확인
3. 데이터베이스 연결 확인
```

### API 키 저장 안 될 때
```
1. API 키 형식 확인
2. 브라우저 LocalStorage 확인
3. 시크릿 모드 아닌지 확인
```
